# HTML Live Editor

HTML 문서를 브라우저에서 바로 수정하고, 페이지별 레이아웃과 웹폰트를 적용할 수 있는 정적 편집기입니다.

## 주요 기능

- HTML 파일 열기, 샘플 로드, 복사, 다운로드
- 프리뷰 화면에서 직접 텍스트 편집
- `data-page`, `data-screen`, `data-slide`, `.page`, `.slide`, `section` 기반 페이지 분류
- 페이지 추가, 이름 변경, 복제, 이동, 삭제
- 16:9 프리뷰 화면
- 제목/본문/포인트 웹폰트와 컬러 설정
- `@font-face` 직접 입력 지원
- 카테고리별 레이아웃 적용
- 기존 제목, 본문, 목록, 이미지 보존형 레이아웃 변환
- 웹 이미지 URL 및 로컬 이미지 적용
- 확장 서식 도구: 굵게, 기울임, H1-H3, 글자 크기, 줄간격, 하이라이트, 링크, 서식 제거

## 레이아웃 카테고리

- 제목: 5개
- 목차: 5개
- 본문 사진 제외: 5개
- 본문 사진 포함: 10개
- 비교: 5개
- 가격 및 견적: 5개
- 내용 요약 및 정리: 5개
- 연락: 5개
- 마무리: 5개

## 실행

정적 파일이므로 `index.html`을 직접 열어도 동작합니다. 로컬 서버로 확인하려면 다음 명령을 사용합니다.

```bash
node server.mjs
```

브라우저에서 아래 주소를 엽니다.

```text
http://127.0.0.1:4189
```

## 검증

검증 스크립트는 Chrome DevTools Protocol에 연결해 주요 동작을 확인합니다.

```bash
node verify.mjs
```

검증 전에 Chrome을 `--remote-debugging-port=9339`로 실행해야 합니다.
